## Play around with Cypress end-to-end testing
### Run test

**Run with GUI**

```
# Go to test path, for example: e2e-google-testing
$ cd path-to/e2e-google-testing
$ yarn
$ ./node_modules/.bin/cypress open
```

**Run with headless browser on docker environment**
```
$ cd path-to/e2e-google-testing
$ /bin/bash ./cy-run.sh
```
